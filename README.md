# Python File Organizer

A Python script that automatically organizes files in a folder into subfolders based on their file types. Users can select any folder on their system using a folder selection dialog (Tkinter), and the script will move files into categories like Images, Documents, Videos, and Audio.

## Features

- Organizes files into category folders automatically  
- Supports Images, Documents, Videos, and Audio  
- Skips duplicate files to prevent overwriting  
- Works on any folder selected by the user  
- User-friendly folder selection dialog using Tkinter  
- Leaves unmatched files and existing folders in place  

## Requirements

- Python 3.x  
- Built-in Python libraries: `os`, `shutil`, `tkinter`
